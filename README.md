# nekos.life RESTful Api
RESTful API for nekos.life with python
Python:3.6
## 安裝 Install
- 安裝套件 Install package
  ```
  python -m pip install -r requirements.txt
  ```
- 安裝nekos.life.py Install nekos.life.py
  ```
  python -m pip install nekos.py-master.zip
  ```
